# Testes Utilizando Cypress

Projeto E2E para validar a aplicação:
`https://ebac-agenda-contatos-tan.vercel.app/`

## Cenários cobertos

- Inclusão de contato
- Alteração de contato
- Remoção de contato

## Requisitos

- Node.js 18+ (recomendado 20+)

## Instalação

```bash
npm install
```

## Execução dos testes

Modo headless:

```bash
npm test
```

Modo interativo:

```bash
npm run cy:open
```
